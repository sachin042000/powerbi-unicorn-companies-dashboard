# Unicorn Companies Power BI Dashboard

## Project Overview
This project is an interactive Power BI dashboard built using Unicorn Companies data.
The dashboard provides insights into company valuations, industries, funding trends, investor participation, and global distribution of unicorn startups.

---

## Objectives
- Analyze unicorn company valuations
- Identify top industries producing unicorn startups
- Track funding trends
- Compare countries and continents by unicorn count
- Explore investor participation across startups

---

## Tools & Technologies Used
- Power BI
- Microsoft Excel / CSV Dataset
- Data Cleaning & Transformation
- DAX Measures
- Data Visualization

---

## Dataset Information
The dataset contains:
- Company Name
- Valuation
- Industry
- Country
- Continent
- Funding
- Investors
- Year Founded
- Date Joined Unicorn Club

---

## Dashboard Features
### Key Insights Included:
- Total Unicorn Companies
- Total Funding Raised
- Industry-wise Analysis
- Country-wise Distribution
- Top Valued Companies
- Investor Analysis
- Year-wise Unicorn Growth
